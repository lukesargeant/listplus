<script src="jquery.listplus.js"></script>

There was once a simple html list that required client side sorting.

Then it needed filtering, then grouping, then some animations and then all that code was a bit of a mess...

Listplus wraps these functions in a lightweight jQueryUI widget.

try:

sortBy: [".lastname"] groupBy: ".lastname" filterBy: ["Riv"] hideList: [".lastname"] sortBy:["reverse .lastname"] $('#mylist').listplus({ 'filterBy': ["Pop","Riv","Sh"], 'sortBy': [function(li) {return $(li).data("listplus-filterscore")}] });
